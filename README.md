# Forge Hello World
## Authenticate with Forge
1. Follow these instructions to get an Atlassian account and API key. 
2. Update the `FORGE_EMAIL` and `FORGE_API_KEY` environment variables in the [dev.nix](/.idx/dev.nix) file and save.
3. Click the `Rebuild Environment` button that appears. 
4. Run `forge create` in the terminal to scaffold a Forge app.
